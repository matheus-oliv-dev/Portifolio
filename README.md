# Portfólio Pessoal - Matheus de Oliveira 🚀

Bem-vindo ao repositório do meu site de portfólio pessoal! 

Este projeto foi construído focado em ser **extremamente leve e performático**, utilizando tecnologias Vanilla (sem frameworks pesados), garantindo um carregamento instantâneo para os visitantes e facilidade de hospedagem.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estruturação semântica.
- **CSS3 (Vanilla)**: Sistema de design premium utilizando cores vibrantes, efeitos de "Glassmorphism" (vidro translúcido) e variáveis para facilitar a manutenção.
- **JavaScript**: Animações suaves de "fade-in" no scroll baseadas em `IntersectionObserver` para melhor performance.

## 🌟 Funcionalidades

- Design responsivo: Adaptável para Mobile e Desktop.
- Interface Dark Mode premium.
- Navegação Suave (Smooth Scroll).
- Integração de links diretos para outras aplicações hospedadas na nuvem (Vercel).
- Links para redes sociais (LinkedIn, GitHub, Email).

## 🚀 Como Executar Localmente

Como o projeto não possui dependências de pacotes ou necessidade de build (como Node.js ou npm), para rodar o site no seu computador basta:

1. Fazer o clone deste repositório:
   ```bash
   git clone https://github.com/matheus-oliv-dev/seu-repositorio-aqui.git
   ```
2. Abrir o arquivo `index.html` em qualquer navegador web de sua preferência.

## 🌐 Deploy

Este projeto é otimizado para a plataforma **Vercel**. Por ser um site puramente estático, o deploy é feito instantaneamente na plataforma, sem necessidade de configuração de comandos de build.
